Welcome to the Information Hub! This is a searchable database for finding NOAA and other data and analysis websites relevant to CEFI. Searches can be conducted to find a wide range of atmospheric and physical, chemical and biological ocean data, including reanalysis and model output. Searches can also be used to locate webtools designed to analyze and visualize data on topics such as climate change and marine heatwaves.

How to use the Information Hub?

Select one or more filters and a list of websites matching all the criteria will be returned. Feedback is welcome. If you want to contribute to the list, please submit a github issue or send an Email!
